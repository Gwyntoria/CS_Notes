# Linux系统指令集

指令的一般格式：

```bash
command [option] [object]
# [option] 和 [object] 可以没有也可以是多个
```

## 1. 目录&文件

### 1.1 显示当前目录

```bash
pwd             # Print current Working Directory
```

### 1.2 显示当前目录下的内容

```bash
ls              # LiSt directory content

ls -a           # 显示所有内容，包含隐藏文件
ls -l           # 以 list 的形式显示目录之下的内容
ls -h           # 以可读性较高的形式显示目录之下的内容
ls -lah         # 可以合并 短选项

ls -lah /opt    # 显示特定目录的内容
```
