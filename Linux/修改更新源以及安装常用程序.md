# 修改更新源

**Notice:**`sources.list` 中的更新源地址如带有`cn`，则修改更新源为非必需

```bash
sudo cp /etc/apt/sources.list /etc/apt/sources.list_backup
sudo gedit /etc/apt/sources.list
```

```bash
sudo apt-get update --fix-missing
```
